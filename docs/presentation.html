REAL-TIME WATER LEVEL MONITORING SYSTEM

Using Ultrasonic Sensor & Arduino

Presented by: Uday V

Course: Engineering Project

Institution: Your College

Date: May 2026

🌊 Problem Statement

  • Water overflow leads to wastage and damage.
  • Manual monitoring is inefficient and unreliable.
  • Lack of real-time awareness in tank systems.
  • Need for a low-cost automated solution.

🎯 Project Objectives

  • Monitor water level in real-time.
  • Display level using LCD interface.
  • Provide alert when threshold is reached.
  • Ensure accuracy and stability.
  • Develop a cost-effective system.

🧩 System Components

  • Arduino Uno: The central brain of the system.
  • HC-SR04 Sensor: Ultrasonic distance measurement.
  • I2C LCD 16x2: Real-time data visualization.
  • Buzzer: Audible alert mechanism.
  • Jumper wires & Breadboard for connections.

⚙️ Working Principle

  • 1. Ultrasonic sensor emits sound waves.
  • 2. Waves reflect from water surface.
  • 3. Time is measured → distance calculated.
  • 4. Distance → water level percentage (%).
  • 5. LCD displays real-time statistics.
  • 6. Buzzer activates at threshold.

📐 Mathematical Model

Distance = (Time × 0.034) / 2
Level (%) = ((Tank Height − Distance) / Tank Height) × 100
  • Threshold Condition:
  • If Level ≥ Threshold → Alert Activated (Buzzer ON)

🔌 Circuit Connections

Pin Mapping:

  • HC-SR04: TRIG → D9 | ECHO → D10
  • I2C LCD: SDA → A4 | SCL → A5
  • Buzzer: Positive → D8
  • Power: VCC (5V) & GND to all components

📊 Results & Conclusion

  • Real-time monitoring successfully achieved.
  • Accurate and stable readings using filtering.
  • Buzzer alert effectively prevents overflow.
  • Conclusion: Efficient, low-cost solution with IoT potential.
