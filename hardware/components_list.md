# Hardware Components List

The following components are required to build the Water Level Monitoring System:

| Component | Quantity | Description |
| :--- | :--- | :--- |
| **Arduino Uno R3** | 1 | Main microcontroller board |
| **HC-SR04 Sensor** | 1 | Ultrasonic distance sensor |
| **I2C LCD 16x2** | 1 | Liquid Crystal Display with I2C module |
| **Piezo Buzzer** | 1 | For audible alerts |
| **Breadboard** | 1 | For prototyping connections |
| **Jumper Wires** | 10+ | Male-to-Male and Male-to-Female |
| **USB Cable** | 1 | For programming the Arduino |

## Optional Components
- **9V Battery / DC Adapter** (For standalone operation)
- **Plastic Housing / Case** (For sensor mounting)
